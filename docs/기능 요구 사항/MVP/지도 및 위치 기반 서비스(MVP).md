### [[기능 요구 사항(MVP)]]
### [[지도 및 위치 기반 서비스]]
## 1️⃣ 위치 기반 지도 표시 & 주변 술집 리스트 (MVP)

| ID      | 우선순위 | 내용                                                                                                                           |
| ------- | ---- | ---------------------------------------------------------------------------------------------------------------------------- |
| **FR1** | MUST | 시스템은 **Kakao Map API 혹은 Naver Map API**를 활용하여, 사용자의 **현재 위치 정보를 받아 지도 상에 표시**하고, 그 주변에 있는 **술집·가게의 위치를 시각적으로 표시**할 수 있어야 한다. |
| **FR2** | MUST | 사용자는 **자신의 현재 위치를 기준으로** 주변에 있는 **술집을 검색**할 수 있어야 한다.                                                                        |
| **FR7** | MUST | 사용자가 **지도 페이지에 처음 진입하면**, 화면(예: 왼쪽 패널)에 **반경 1km 내에 있는 주점·이자카야 목록을 추천 기준에 따라 리스트 형태로** 볼 수 있어야 한다.                           |

---

## 2️⃣ 길찾기 & 경로 정보 (MVP)

| ID      | 우선순위 | 내용                                                                     |
| ------- | ---- | ---------------------------------------------------------------------- |
| **FR4** | MUST | 시스템은 **추천된 술집까지의 길찾기 기능**을 제공해야 하며, 사용자가 선택한 술집까지의 **최적 경로를 안내**해야 한다. |

---

## 3️⃣ 지도 인터랙션 & 사용자 편의 (MVP)

|ID|우선순위|내용|
|---|---|---|
|**FR8**|MUST|사용자는 **현재 내 위치로 지도를 이동시키는 상호작용 버튼(예: “내 위치로 이동”)**을 사용할 수 있어야 하며, 이 버튼을 통해 **한 번에 자신의 위치 중심으로 지도 화면을 재배치**할 수 있어야 한다.|
